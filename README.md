# Backend del Proyecto: Portafolio Personal

Este es el repositorio del backend para el proyecto de Portafolio Personal. El backend está construido utilizando Spring Boot y se encarga de gestionar la lógica de negocio, incluyendo la gestión de proyectos y la conexión con la base de datos PostgreSQL

## Estructura del Proyecto

- `/src`: Contiene el código fuente del backend.
- `/src/main/java`: Aquí encontrarás las clases de Java que conforman la lógica del backend.
- `/src/main/resources`: Contiene la configuración y recursos necesarios para la aplicación.

## Configuración

Asegúrate de tener las siguientes dependencias y herramientas instaladas:

- **Java Development Kit (JDK) 17:** Asegúrate de tener JDK 17 instalado en tu sistema.
- **Maven:** El proyecto se gestiona con Maven para la administración de dependencias y construcción.
- Spring Boot
- PostgreSQL

## Instrucciones de Uso

1. Clona este repositorio en tu máquina local.
2. Abre el proyecto en tu entorno de desarrollo.
3. Configura la conexión a tu base de datos PostgreSQL en `application.properties`.
4. Ejecuta la aplicación Spring Boot utilizando Maven:

mvn spring-boot:run

## Endpoints API

El backend proporciona una serie de endpoints API para interactuar con el proyecto. Estos incluyen:

- CRUD de proyectos.
- Gestión de datos de contacto.
- Otras funcionalidades relacionadas con el proyecto.
